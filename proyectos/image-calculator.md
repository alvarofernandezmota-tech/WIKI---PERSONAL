# 🐍 image-calculator — OCR + Calculadora

> Repo: [alvarofernandezmota-tech/image-calculator](https://github.com/alvarofernandezmota-tech/image-calculator)
> Creado: abril 2026 · Estado: **estable / mantenimiento mínimo**

## Qué es

App Python que lee números en imágenes mediante OCR y los opera. GUI con tkinter. Arquitectura modular (núcleo separado de interfaz).

## Stack

- Python + tkinter (GUI)
- OCR: Tesseract / pytesseract
- Operaciones: suma, resta, multiplicación, división, comparación

## Valor

- Primer proyecto Python con GUI real
- Primer uso de OCR
- Arquitectura modular bien separada

## Relación con el ecosistema

Proyecto de aprendizaje. No conecta con [[proyectos/thdora]] ni con [[proyectos/ai-toolkit]] directamente.
