## Qué es este repo

`yggdrasil-dew` es el repositorio central del ecosistema Yggdrasil. Aquí vive:

- La documentación real del sistema
- Los diarios y sesiones de trabajo
- Las specs de los agentes IA (cuando estén escritas de verdad)
- Los scripts operativos
- El inbox para entrada de tareas e ideas

## Qué NO es este repo

| Cosa | Dónde va (cuando exista) |
|---|---|
| Código de producción de agentes | Su propio repo (`yggdrasil-secops`, etc.) |
| Infraestructura real (Docker en producción) | `infra/` aquí, o repo propio si crece |
| Cosas personales (vida, finanzas, salud) | Repo separado privado |
| OSINT, bots Telegram, RAG local | Sus propios repos cuando existan |
