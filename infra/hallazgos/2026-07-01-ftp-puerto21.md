# Hallazgo: Puerto 21 FTP Abierto en Router

**Fecha:** 2026-07-01  
**Severidad:** 🔴 ALTA  
**Descubierto en:** Pentest inicial Fase 3  
**Herramienta:** nmap

---

## Descripción

Durante el escaneo de la red local 192.168.1.0/24, se detectó el puerto 21 (FTP) abierto en el router doméstico.

## Riesgo

- FTP transmite credenciales en **texto plano**
- Si el router tiene FTP habilitado internamente, cualquier dispositivo en la LAN puede intentar autenticarse
- Potencial vector de acceso si credenciales son débiles o por defecto

## Evidencia

```
nmap -sV 192.168.1.1
PORT   STATE SERVICE VERSION
21/tcp open  ftp
```

## Mitigación

1. Acceder al panel de administración del router
2. Desactivar servicio FTP si no se usa
3. Si se necesita, migrar a SFTP/FTPS
4. Cambiar credenciales por defecto del router
