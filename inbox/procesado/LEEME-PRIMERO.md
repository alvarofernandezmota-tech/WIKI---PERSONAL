# LEEME PRIMERO — Cómo usar este inbox

> **El inbox es la bandeja de entrada del ecosistema.**
> Todo lo que llega sin clasificar va aquí. Nada más.

---

## Reglas

1. **Entra todo** — ideas, notas de sesiones, pendientes, decisiones, capturas de contexto
2. **No se organiza al vuelo** — entra rápido, se clasifica después
3. **Nombre de archivo:** `YYYY-MM-DD-descripcion-corta.md`
4. **Cuando se clasifica** — el archivo se mueve a su carpeta definitiva y se borra de inbox
