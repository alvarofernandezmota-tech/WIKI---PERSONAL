# Inbox sesiones

Aquí se guarda el contexto resumido de cada sesión de trabajo:
- Decisiones tomadas
- Tareas abiertas al cierre
- Bloqueos encontrados
- Siguiente paso recomendado

Formato de nombre: `sesion-YYYY-MM-DD.md`
Complementa `diary/` (que tiene el resumen técnico detallado).
No sustituye al diary, sino que añade el contexto humano de continuidad.
