# agente-roadmap-master

## Función única
Centralizar todos los pendientes del ecosistema y generar/actualizar el issue maestro en GitHub.

## Entradas
- Pendientes de todos los demás agentes
- Entradas de `inbox/proyectos/`
- Deuda técnica detectada por auditores

## Salidas
- Issue maestro actualizado en GitHub
- Roadmap en `docs/roadmap/`
- Traza en `inbox/_meta/`

## Relación con otros agentes
- Recibe pendientes de todos los agentes
- Es el punto de entrada para el humano en cada sesión
