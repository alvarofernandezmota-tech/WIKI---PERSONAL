# 📥 Inbox

Notas rápidas de sesiones. Se procesan periódicamente y se mueven a su sitio correcto.

## Regla
- Aquí llegan notas rápidas sin clasificar
- Una vez a la semana se revisan y mueven a su carpeta correcta
- El inbox debe quedar vacío después de procesar

## Cómo crear una nota rápida

```bash
nano ~/repos/WIKI---PERSONAL/inbox/$(date +%Y-%m-%d)-nota.md
```
