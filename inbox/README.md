# 📥 Inbox — El Baúl

> Todo lo nuevo entra aquí primero. Sin organizar, sin pensar dónde va.
> Cuando una nota tenga destino claro → se mueve a su carpeta.

---

## Reglas del baúl

1. **Entra todo** — ideas, comandos, snippets, notas sueltas, cosas a revisar
2. **No se organiza aquí** — no pongas subcarpetas dentro de inbox
3. **Se vacía periódicamente** — cada domingo revisa qué tiene destino y muévelo
4. **Tags mínimos** — al menos uno: `#thdora` `#infra` `#python` `#osint` `#personal` `#revisar`

## Formato de nota rápida

```md
---
tags: [revisar, thdora]
fecha: YYYY-MM-DD
---

Lo que sea que necesites recordar.
```

## Destinos posibles al mover

| Tag frecuente | Destino |
|---|---|
| `#thdora` `#infra` | `proyectos/thdora.md` o `setup/` |
| `#python` `#osint` | `formacion/` |
| `#personal` | `yo/` |
| `#idea` | `proyectos/ideas.md` |
