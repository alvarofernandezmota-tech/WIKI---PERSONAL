# 🇪🇺 Mistral Large 2 — Alternativa Soberana para Código y Estructuras Multilingües

## Modelo y versión
- **Identificador:** mistral-large-latest / Mistral Large 2
- **Versión:** Estable

## Empresa y lanzamiento
- **Desarrollador:** Mistral AI (París, Francia)
- **Fecha de lanzamiento:** Consolidado en el mercado de modelos comerciales de frontera europeos

## Arquitectura y entrenamiento
- **Tipo:** Transformador denso optimizado para alta eficiencia coste-rendimiento
- **Enfoque:** Fuertemente alineado en programación multilingüe, razonamiento lógico y adhesión estricta a esquemas de salida sintáctica formateada

## Ventana de contexto
- **Capacidad:** 128.000 tokens

## Modalidades
- **Entrada:** Texto, Código estructurado
- **Salida:** Texto, Código, JSON nativo preciso

## Herramientas nativas
- **Function Calling Avanzado:** Diseñado desde su arquitectura para interconexión modular y ejecución precisa de llamadas a APIs nativas
- **JSON Estricto:** Fuerza respuestas que se adhieren rigurosamente a un esquema proporcionado

## Modos especiales
- **Alineamiento de JSON Estricto:** Output garantizado en esquema definido por el desarrollador
- **Le Chat:** Interfaz web con acceso a búsqueda web y canvas de documentos

## Benchmarks (junio 2026)
- **HumanEval / MBPP:** Puntuaciones robustas en generación y corrección de scripts Python, SQL y C++
- **Multilingual benchmarks:** Top en rendimiento multilingüe europeo

## Mejor para
- Canalizaciones de automatización que requieran salida de datos extremadamente estructurada
- Procesamiento técnico multilingüe dentro del marco normativo europeo (GDPR)
- Pipelines de Function Calling precisos y repetitivos

## Peor para (usar otra IA)
- Análisis multimodal avanzado basado en imágenes complejas o vídeo
- Tareas agénticas de control directo de interfaces de usuario (Computer Use)
- Contexto masivo >128K tokens

## Coste

- **Auto-alojado:** Disponible con licencias específicas en clouds asociadas
