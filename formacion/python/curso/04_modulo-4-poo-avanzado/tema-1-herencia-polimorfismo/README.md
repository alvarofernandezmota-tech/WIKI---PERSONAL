# Tema 1 - Herencia y Polimorfismo

## 🎯 Objetivos

- Comprender y aplicar herencia simple y múltiple
- Entender el Method Resolution Order (MRO)
- Implementar polimorfismo en Python
- Dominar el uso de `super()` e `issubclass()`

## 📚 Contenido

### 1.1 Herencia
Concepto fundamental de POO que permite crear clases basadas en clases existentes.

### 1.2 Herencia Simple
Una clase hija hereda de una única clase padre.

### 1.3 Herencia Múltiple
Una clase hija hereda de múltiples clases padre.

### 1.4 Method Resolution Order (MRO)
Orden en que Python busca métodos en la jerarquía de herencia.

### 1.5 Polimorfismo
Capacidad de diferentes clases de responder al mismo método de manera distinta.

### 1.6 Uso de super()
Función para llamar métodos de la clase padre.

### 1.7 Uso de issubclass()
Verificar si una clase es subclase de otra.
