# Ejercicios del Módulo 4

## 🎯 Descripción

Ejercicios prácticos para consolidar los conocimientos del Módulo 4.

## 📝 Ejercicios

### Ejercicio 1: Sistema de Vehículos
Implementar un sistema de vehículos usando herencia y polimorfismo.

**Requisitos:**
- Clase base `Vehiculo`
- Clases hijas: `Coche`, `Moto`, `Camion`
- Herencia múltiple para vehículos eléctricos
- Polimorfismo en cálculo de impuestos

### Ejercicio 2: Sistema de Pagos (ABC)
Crear un sistema de pagos usando interfaces.

**Requisitos:**
- Interface `MetodoPago` (ABC)
- Implementaciones: `TarjetaCredito`, `PayPal`, `Bitcoin`
- Métodos abstractos: `procesar_pago()`, `validar()`

### Ejercicio 3: Gestor de Eventos
Sistema de gestión de eventos con fechas.

**Requisitos:**
- Clase `Evento` con datetime
- Calcular duración con timedelta
- Manejar zonas horarias (aware)
- Validar fechas futuras

### Ejercicio 4: Generador de Reportes
Generador para procesar grandes cantidades de datos.

**Requisitos:**
- Generador que lea archivo línea por línea
- Filtrar datos con expresiones generadoras
- Optimizar uso de memoria

### Ejercicio 5: Sistema Bancario con Excepciones
Sistema bancario con manejo robusto de errores.

**Requisitos:**
- Excepciones personalizadas: `SaldoInsuficiente`, `CuentaBloqueada`
- Try-except-else-finally
- Uso de assert para validaciones

### Ejercicio 6: Tests para Calculadora
Escribir tests completos para una calculadora.

**Requisitos:**
- Tests con pytest
- Fixtures para setup
- Tests parametrizados
- Cobertura >80%

## 📚 Recursos

- PDFs de ejercicios en carpeta `ejercicios-pdf/`
- Soluciones en carpeta `soluciones/`
