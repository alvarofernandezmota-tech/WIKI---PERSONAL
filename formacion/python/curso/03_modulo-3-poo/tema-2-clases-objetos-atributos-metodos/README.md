## 💡 Conceptos Clave

```python
# Clase: Plantilla/Molde
class Coche:
    pass

# Objeto: Instancia concreta
mi_coche = Coche()

# Atributo: Característica
mi_coche.color = "rojo"

# Método: Acción
mi_coche.arrancar()
```
