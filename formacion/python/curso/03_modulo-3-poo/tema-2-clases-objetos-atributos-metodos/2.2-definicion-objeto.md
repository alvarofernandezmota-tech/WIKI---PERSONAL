# 2.2 Definición de Objeto

## 🎯 ¿Qué es un Objeto?

**Un objeto es una instancia concreta de una clase.**

Analogía:
- **Clase Galleta**: Molde de galleta 🍪
- **Objeto**: Cada galleta hecha con ese molde 🍪🍪🍪

Cada galleta (objeto) tiene las mismas características (forma, tamaño) pero puede tener valores diferentes (sabor, decoración).

---

## 🛠️ Crear Objetos (Instanciar)

### Sintaxis:
```python
nombre_objeto = NombreClase(argumentos)
```

### Ejemplo:
```python
class Perro:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad
    
    def ladrar(self):
        return f"{self.nombre} dice: Guau!"

# Crear objetos (instancias)
perro1 = Perro("Max", 3)
perro2 = Perro("Luna", 5)
perro3 = Perro("Rocky", 2)

print(perro1.nombre)  # Max
print(perro2.nombre)  # Luna
print(perro3.ladrar())  # Rocky dice: Guau!
```

---

## 🔢 Múltiples Instancias

**Una clase puede generar infinitos objetos.**

```python
class Coche:
    def __init__(self, marca, color):
        self.marca = marca
        self.color = color

# Crear varios coches
coche1 = Coche("Toyota", "rojo")
coche2 = Coche("Honda", "azul")
coche3 = Coche("Ford", "negro")

print(coche1.marca, coche1.color)  # Toyota rojo
print(coche2.marca, coche2.color)  # Honda azul
print(coche3.marca, coche3.color)  # Ford negro
```

**Cada objeto es independiente:**
```python
coche1.marca = "Mazda"  # Solo cambia coche1
print(coche1.marca)  # Mazda
print(coche2.marca)  # Honda (no cambió)
```

## 🎯 Identidad de Objetos

### Cada objeto tiene una identidad única

```python
class Persona:
    def __init__(self, nombre):
        self.nombre = nombre

persona1 = Persona("Ana")
persona2 = Persona("Ana")
persona3 = persona1  # Referencia al mismo objeto

# Comparar identidad con 'is'
print(persona1 is persona2)  # False (objetos diferentes)
print(persona1 is persona3)  # True (mismo objeto)

# Comparar valores con '=='
print(persona1.nombre == persona2.nombre)  # True (mismo nombre)

# Ver ID único
print(id(persona1))  # Ej: 140234567890
print(id(persona2))  # Ej: 140234567920 (diferente)
print(id(persona3))  # Ej: 140234567890 (igual que persona1)
```

## 📊 Objetos en Acción

### Ejemplo: Sistema de Estudiantes

```python
class Estudiante:
    def __init__(self, nombre, edad, carrera):
        self.nombre = nombre
        self.edad = edad
        self.carrera = carrera
        self.calificaciones = []
    
    def agregar_calificacion(self, calificacion):
        if 0 <= calificacion <= 10:
            self.calificaciones.append(calificacion)
    
    def promedio(self):
        if self.calificaciones:
            return sum(self.calificaciones) / len(self.calificaciones)
        return 0
    
    def info(self):
        return f"{self.nombre}, {self.edad} años - {self.carrera}"

# Crear varios estudiantes
estudiante1 = Estudiante("Ana", 20, "Ingeniería")
estudiante2 = Estudiante("Juan", 22, "Medicina")
estudiante3 = Estudiante("María", 21, "Derecho")

# Cada uno tiene sus propias calificaciones
estudiante1.agregar_calificacion(8.5)
estudiante1.agregar_calificacion(9.0)
estudiante1.agregar_calificacion(7.5)

estudiante2.agregar_calificacion(9.5)
estudiante2.agregar_calificacion(9.0)

print(estudiante1.info())
print(f"Promedio: {estudiante1.promedio():.2f}")
print()
print(estudiante2.info())
print(f"Promedio: {estudiante2.promedio():.2f}")
```

## 📦 Objetos como Parámetros

**Puedes pasar objetos a funciones:**

```python
class Producto:
    def __init__(self, nombre, precio):
        self.nombre = nombre
        self.precio = precio

def aplicar_descuento(producto, porcentaje):
    """Aplica descuento a un producto"""
    descuento = producto.precio * (porcentaje / 100)
    producto.precio -= descuento
    return f"Nuevo precio de {producto.nombre}: {producto.precio:.2f}€"

# Crear producto
laptop = Producto("Laptop HP", 899.99)
print(f"Precio original: {laptop.precio}€")

# Aplicar descuento
print(aplicar_descuento(laptop, 15))  # 15% descuento
```

## 📝 Listas de Objetos

**Puedes almacenar objetos en listas:**

```python
class Tarea:
    def __init__(self, titulo, completada=False):
        self.titulo = titulo
        self.completada = completada
    
    def completar(self):
        self.completada = True
    
    def __str__(self):
        estado = "✅" if self.completada else "❌"
        return f"{estado} {self.titulo}"

# Crear lista de tareas
tareas = [
    Tarea("Estudiar Python"),
    Tarea("Hacer ejercicios POO"),
    Tarea("Leer apuntes")
]

# Completar primera tarea
tareas[0].completar()

# Mostrar todas
for tarea in tareas:
    print(tarea)
```

## 🧙 Objetos Anidados

**Objetos pueden contener otros objetos:**

```python
class Direccion:
    def __init__(self, calle, ciudad, codigo_postal):
        self.calle = calle
        self.ciudad = ciudad
        self.codigo_postal = codigo_postal
    
    def __str__(self):
        return f"{self.calle}, {self.ciudad} ({self.codigo_postal})"

class Persona:
    def __init__(self, nombre, direccion):
        self.nombre = nombre
        self.direccion = direccion  # Objeto dentro de objeto
    
    def info_completa(self):
        return f"{self.nombre} vive en {self.direccion}"

# Crear objetos
direccion_ana = Direccion("Calle Mayor 5", "Madrid", "28013")
persona = Persona("Ana", direccion_ana)

print(persona.info_completa())
print(f"Ciudad: {persona.direccion.ciudad}")  # Acceso anidado
```

## 🔍 Clase vs Objeto vs Instancia

```python
# CLASE: Definición/Molde
class Coche:
    def __init__(self, marca):
        self.marca = marca

# OBJETO/INSTANCIA: Implementación concreta
mi_coche = Coche("Toyota")  # Objeto 1
tu_coche = Coche("Honda")   # Objeto 2

print(type(mi_coche))  # <class '__main__.Coche'>
print(isinstance(mi_coche, Coche))  # True
```

**Terminología:**
- **Clase**: El concepto abstracto
- **Objeto**: La cosa concreta
- **Instancia**: Sinónimo de objeto
- **Instanciar**: Crear un objeto desde una clase

---

## ✅ Verificar Tipo de Objeto

```python
class Perro:
    pass

class Gato:
    pass

mi_perro = Perro()
mi_gato = Gato()

# type() - Devuelve la clase
print(type(mi_perro))  # <class '__main__.Perro'>

# isinstance() - Verifica si es instancia
print(isinstance(mi_perro, Perro))  # True
print(isinstance(mi_perro, Gato))   # False

# Verificar múltiples tipos
print(isinstance(mi_perro, (Perro, Gato)))  # True (es Perro)
```

---

## 📝 Resumen

- **Objeto** = Instancia concreta de una clase
- **Crear objeto**: `objeto = Clase(argumentos)`
- **Cada objeto** es independiente
- **Identidad**: Cada objeto tiene ID único
- **Múltiples instancias**: Puedes crear cuantas quieras
- **Objetos en estructuras**: Listas, diccionarios, etc.

### Analogía final:
```
Clase Coche 🛠️  = Planos del coche
Objeto coche1 🚗 = Toyota Corolla rojo
Objeto coche2 🚙 = Honda Civic azul
Objeto coche3 🚘 = Ford Focus negro
```
