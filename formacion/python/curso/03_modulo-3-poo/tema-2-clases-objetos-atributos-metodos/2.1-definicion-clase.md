# 2.1 Definición de Clase

## 🎯 ¿Qué es una Clase?

**Una clase es un modelo o plantilla para crear objetos.**

Es como un plano arquitectónico:
- El **plano** define cómo será una casa (clase)
- La **casa construida** es el objeto real (instancia)

---

## 📝 Sintaxis Básica

```python
class NombreClase:
    """Docstring de la clase"""
    # Atributos y métodos aquí
    pass
```

### Ejemplo simple:
```python
class Perro:
    """Clase que representa un perro"""
    pass

# Crear una instancia
mi_perro = Perro()
print(type(mi_perro))  # <class '__main__.Perro'>
```

---

## 📜 Convenciones de Nombres

### Correcto:
```python
class Coche:        # ✅ CamelCase
class PersonaAdulta: # ✅ CamelCase
class Usuario:      # ✅ CamelCase
```

### Incorrecto:
```python
class mi_clase:     # ❌ snake_case (para funciones)
class MICLASE:      # ❌ MAYUSCULAS (para constantes)
class miClase:      # ❌ camelCase (no se usa en Python)
```

---

## 🔍 Clases vs Funciones

### Función (programación estructurada):
```python
def crear_persona(nombre, edad):
    return {'nombre': nombre, 'edad': edad}

def saludar_persona(persona):
    return f"Hola, soy {persona['nombre']}"

# Uso
persona1 = crear_persona("Ana", 25)
print(saludar_persona(persona1))
```

### Clase (POO):
```python
class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad
    
    def saludar(self):
        return f"Hola, soy {self.nombre}"

# Uso
persona1 = Persona("Ana", 25)
print(persona1.saludar())
```

**Ventaja de la clase:** Datos y comportamiento juntos 🔗

## 📊 Anatomía de una Clase

```python
class Coche:
    """Clase que representa un coche"""
    
    # Atributo de clase (compartido por todas las instancias)
    ruedas = 4
    
    # Constructor
    def __init__(self, marca, modelo, color):
        # Atributos de instancia (propios de cada objeto)
        self.marca = marca
        self.modelo = modelo
        self.color = color
        self.velocidad = 0
    
    # Método de instancia
    def acelerar(self, incremento):
        self.velocidad += incremento
        return f"Velocidad actual: {self.velocidad} km/h"
    
    def describir(self):
        return f"{self.marca} {self.modelo} de color {self.color}"
```

### Uso:
```python
# Crear instancia
mi_coche = Coche("Toyota", "Corolla", "rojo")

# Acceder a atributos
print(mi_coche.marca)  # Toyota
print(mi_coche.ruedas)  # 4

# Llamar métodos
print(mi_coche.describir())
print(mi_coche.acelerar(50))
```

## 🎨 Ejemplo: Clase Cuenta Bancaria

```python
class CuentaBancaria:
    """Representa una cuenta bancaria simple"""
    
    def __init__(self, titular, saldo_inicial=0):
        self.titular = titular
        self.saldo = saldo_inicial
        self.movimientos = []
    
    def depositar(self, cantidad):
        if cantidad > 0:
            self.saldo += cantidad
            self.movimientos.append(f"Depósito: +{cantidad}€")
            return True
        return False
    
    def retirar(self, cantidad):
        if 0 < cantidad <= self.saldo:
            self.saldo -= cantidad
            self.movimientos.append(f"Retiro: -{cantidad}€")
            return True
        return False
    
    def ver_saldo(self):
        return f"Saldo de {self.titular}: {self.saldo}€"
    
    def historial(self):
        return '\n'.join(self.movimientos)

# Uso
cuenta = CuentaBancaria("Ana", 1000)
cuenta.depositar(500)
cuenta.retirar(200)
print(cuenta.ver_saldo())
print(cuenta.historial())
```

## 📝 Plantilla para Crear Clases

```python
class MiClase:
    """
    Descripción breve de la clase.
    
    Attributes:
        atributo1 (tipo): Descripción
        atributo2 (tipo): Descripción
    """
    
    # Atributos de clase (si hay)
    atributo_clase = valor
    
    def __init__(self, parametro1, parametro2):
        """
        Constructor de la clase.
        
        Args:
            parametro1 (tipo): Descripción
            parametro2 (tipo): Descripción
        """
        self.atributo1 = parametro1
        self.atributo2 = parametro2
    
    def metodo1(self, parametro):
        """
        Descripción del método.
        
        Args:
            parametro (tipo): Descripción
        
        Returns:
            tipo: Descripción del retorno
        """
        # Código del método
        pass
```

## ✅ Checklist: ¿Está bien mi clase?

- [ ] Nombre en CamelCase
- [ ] Tiene docstring
- [ ] Tiene constructor `__init__`
- [ ] Atributos de instancia inician con `self.`
- [ ] Métodos tienen `self` como primer parámetro
- [ ] Representa un concepto claro
- [ ] Responsabilidad única (hace una cosa bien)

---

## 💡 Cuándo Crear una Clase

### Crea una clase cuando:
- ✅ Tienes datos y comportamiento relacionados
- ✅ Necesitas crear múltiples objetos similares
- ✅ Quieres organizar tu código
- ✅ Necesitas mantener estado

### NO necesitas clase cuando:
- ❌ Solo tienes funciones sin estado
- ❌ Una función simple resuelve el problema
- ❌ Estás complicando innecesariamente

---

## 📝 Resumen

- **Clase** = Plantilla/Molde para crear objetos
- **Sintaxis**: `class NombreClase:`
- **Convención**: CamelCase
- **Contiene**: Atributos y métodos
- **Ventaja**: Organiza datos y comportamiento juntos
