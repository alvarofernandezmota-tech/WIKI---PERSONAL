### 1.2 Los 4 Principios de la POO
1. **Encapsulación**: Ocultar datos internos
2. **Abstracción**: Mostrar solo lo esencial
3. **Herencia**: Reutilizar código
4. **Polimorfismo**: Múltiples formas
