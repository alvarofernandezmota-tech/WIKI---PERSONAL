# 3.1 Constructores

## 🎯 ¿Qué es un Constructor?

El **constructor** es un método especial que se ejecuta automáticamente cuando creas una nueva instancia de una clase.

En Python, el constructor se llama `__init__`.

---

## 📝 Sintaxis Básica

```python
class MiClase:
    def __init__(self, parametro1, parametro2):
        self.atributo1 = parametro1
        self.atributo2 = parametro2
```

---

## 🔍 Componentes del Constructor

### **1. `__init__`**
```python
def __init__(self):
    # Código de inicialización
```
- Doble guion bajo antes y después
- Se pronuncia "dunder init" (double underscore init)
- Es un método mágico de Python

### **2. `self`**
```python
def __init__(self, nombre):
    self.nombre = nombre
    #    ↑      ↑
    # Atributo  Parámetro
```
- Primer parámetro siempre es `self`
- Representa la instancia actual
- Python lo pasa automáticamente

### **3. Parámetros**
```python
def __init__(self, nombre, edad, ciudad="Madrid"):
    self.nombre = nombre
    self.edad = edad
    self.ciudad = ciudad  # Valor por defecto
```

## 💡 Ejemplos Progresivos

### **Nivel 1: Constructor Simple**
```python
class Usuario:
    def __init__(self, username):
        self.username = username

# Uso
u = Usuario("alvaro")
print(u.username)  # alvaro
```

### **Nivel 2: Múltiples Parámetros**
```python
class Usuario:
    def __init__(self, username, email, rol):
        self.username = username
        self.email = email
        self.rol = rol
    
    def descripcion(self):
        return f"{self.username} ({self.rol}) - {self.email}"

# Uso
u = Usuario("alvaro", "alvaro@example.com", "admin")
print(u.descripcion())
```

### **Nivel 3: Valores por Defecto**
```python
class Usuario:
    def __init__(self, username, email, rol="user", activo=True):
        self.username = username
        self.email = email
        self.rol = rol
        self.activo = activo

# Uso
u1 = Usuario("alvaro", "alvaro@example.com")  # rol="user", activo=True
u2 = Usuario("admin", "admin@example.com", "admin")  # activo=True
u3 = Usuario("test", "test@example.com", "user", False)  # Todo especificado
```

### **Nivel 4: Validaciones**
```python
class Usuario:
    def __init__(self, username, email, edad):
        # Validar username
        if not username or len(username) < 3:
            raise ValueError("Username debe tener al menos 3 caracteres")
        
        # Validar email
        if "@" not in email:
            raise ValueError("Email inválido")
        
        # Validar edad
        if not isinstance(edad, int) or edad < 0 or edad > 120:
            raise ValueError("Edad debe estar entre 0 y 120")
        
        self.username = username
        self.email = email
        self.edad = edad

# Uso
u1 = Usuario("alvaro", "alvaro@example.com", 25)  # ✅ OK
u2 = Usuario("al", "test@example.com", 25)        # ❌ ValueError
u3 = Usuario("alvaro", "invalido", 25)            # ❌ ValueError
```

## 🔄 Constructores Alternativos con `@classmethod`

### **Problema:**
```python
class Configuracion:
    def __init__(self, host, puerto, db_name):
        self.host = host
        self.puerto = puerto
        self.db_name = db_name

# Crear desde diccionario es complicado
config_dict = {"host": "localhost", "puerto": 5432, "db_name": "mydb"}
config = Configuracion(config_dict["host"], config_dict["puerto"], config_dict["db_name"])  # ❌ Feo
```

### **Solución con `@classmethod`:**
```python
class Configuracion:
    def __init__(self, host, puerto, db_name):
        self.host = host
        self.puerto = puerto
        self.db_name = db_name
    
    @classmethod
    def desde_dict(cls, config_dict):
        """Constructor alternativo desde diccionario"""
        return cls(
            config_dict["host"],
            config_dict["puerto"],
            config_dict["db_name"]
        )
    
    @classmethod
    def desarrollo(cls):
        """Constructor para entorno desarrollo"""
        return cls("localhost", 5432, "dev_db")
    
    @classmethod
    def produccion(cls):
        """Constructor para entorno producción"""
        return cls("prod-server.com", 5432, "prod_db")

# Uso
config1 = Configuracion("localhost", 5432, "mydb")  # Constructor normal

config_dict = {"host": "localhost", "puerto": 5432, "db_name": "mydb"}
config2 = Configuracion.desde_dict(config_dict)  # Desde dict ✅

config3 = Configuracion.desarrollo()  # Entorno desarrollo ✅
config4 = Configuracion.produccion()  # Entorno producción ✅
```

## 🎨 Ejemplo Real: Sistema de Base de Datos

```python
from datetime import datetime
import hashlib

class ConexionDB:
    """Representa una conexión a base de datos"""
    
    conexiones_activas = 0  # Contador de clase
    
    def __init__(self, host, puerto, database, usuario, password):
        # Validaciones
        if not host:
            raise ValueError("Host es obligatorio")
        if not isinstance(puerto, int) or puerto < 1 or puerto > 65535:
            raise ValueError("Puerto debe estar entre 1 y 65535")
        
        # Asignación
        self.host = host
        self.puerto = puerto
        self.database = database
        self.usuario = usuario
        self._password_hash = self._hashear_password(password)
        self.conectado = False
        self.creada_en = datetime.now()
        
        # Incrementar contador
        ConexionDB.conexiones_activas += 1
    
    @staticmethod
    def _hashear_password(password):
        """Hashea la contraseña (método auxiliar)"""
        return hashlib.sha256(password.encode()).hexdigest()
    
    @classmethod
    def desde_url(cls, url):
        """Constructor desde URL de conexión"""
        # Parsear URL: postgresql://usuario:pass@host:puerto/database
        partes = url.replace("postgresql://", "").split("@")
        credenciales = partes[0].split(":")
        servidor = partes[1].split("/")
        host_puerto = servidor[0].split(":")
        
        return cls(
            host=host_puerto[0],
            puerto=int(host_puerto[1]),
            database=servidor[1],
            usuario=credenciales[0],
            password=credenciales[1]
        )
    
    @classmethod
    def localhost_postgres(cls, database):
        """Constructor para PostgreSQL local"""
        return cls("localhost", 5432, database, "postgres", "postgres")
    
    @classmethod
    def total_conexiones(cls):
        """Retorna total de conexiones activas"""
        return cls.conexiones_activas
    
    def conectar(self):
        self.conectado = True
        print(f"Conectado a {self.database} en {self.host}:{self.puerto}")
    
    def desconectar(self):
        self.conectado = False
        ConexionDB.conexiones_activas -= 1
        print(f"Desconectado de {self.database}")
    
    def __del__(self):
        """Destructor - se ejecuta al eliminar objeto"""
        if self.conectado:
            self.desconectar()

# Uso
# Constructor normal
db1 = ConexionDB("localhost", 5432, "mydb", "user", "pass123")

# Constructor desde URL
db2 = ConexionDB.desde_url("postgresql://admin:secret@prod-server.com:5432/production")

# Constructor simplificado
db3 = ConexionDB.localhost_postgres("test_db")

print(ConexionDB.total_conexiones())  # 3

db1.conectar()
db2.conectar()
db3.conectar()
```

## ✅ Checklist Constructor

- [ ] Nombre correcto: `__init__`
- [ ] Primer parámetro: `self`
- [ ] Inicializa todos los atributos necesarios
- [ ] Validaciones antes de asignar
- [ ] Valores por defecto cuando tenga sentido
- [ ] Documentado con docstring
- [ ] Constructores alternativos con `@classmethod` si son útiles

## 🔧 Ejemplo Técnico: Logger Personalizado

```python
import logging
from datetime import datetime
from pathlib import Path

class Logger:
    """Sistema de logging personalizado"""
    
    _instancia = None  # Singleton
    
    def __init__(self, nombre, nivel=logging.INFO, archivo=None):
        self.nombre = nombre
        self.nivel = nivel
        self.archivo = archivo
        self._logger = self._configurar_logger()
    
    def _configurar_logger(self):
        """Configura el logger de Python"""
        logger = logging.getLogger(self.nombre)
        logger.setLevel(self.nivel)
        
        # Handler para consola
        console_handler = logging.StreamHandler()
        console_handler.setLevel(self.nivel)
        
        # Formato
        formato = logging.Formatter(
            '%(asctime)s - %(name)s - %(levelname)s - %(message)s'
        )
        console_handler.setFormatter(formato)
        logger.addHandler(console_handler)
        
        # Handler para archivo (si se especifica)
        if self.archivo:
            file_handler = logging.FileHandler(self.archivo)
            file_handler.setFormatter(formato)
            logger.addHandler(file_handler)
        
        return logger
    
    @classmethod
    def get_instance(cls, nombre="app"):
        """Patrón Singleton - una sola instancia"""
        if cls._instancia is None:
            cls._instancia = cls(nombre)
        return cls._instancia
    
    @classmethod
    def para_desarrollo(cls):
        """Logger para desarrollo (DEBUG)"""
        return cls("dev", logging.DEBUG)
    
    @classmethod
    def para_produccion(cls, archivo_log):
        """Logger para producción (WARNING+)"""
        return cls("prod", logging.WARNING, archivo_log)
    
    def debug(self, mensaje):
        self._logger.debug(mensaje)
    
    def info(self, mensaje):
        self._logger.info(mensaje)
    
    def warning(self, mensaje):
        self._logger.warning(mensaje)
    
    def error(self, mensaje):
        self._logger.error(mensaje)

# Uso
logger_dev = Logger.para_desarrollo()
logger_dev.debug("Mensaje de depuración")
logger_dev.info("Aplicación iniciada")

logger_prod = Logger.para_produccion("app.log")
logger_prod.warning("Advertencia en producción")
logger_prod.error("Error crítico")
```

## 📝 Resumen

- **Constructor** = Método `__init__` que inicializa objetos
- **`self`** = Referencia a la instancia actual
- **Validaciones** = Siempre en el constructor
- **Valores por defecto** = Para parámetros opcionales
- **`@classmethod`** = Constructores alternativos
