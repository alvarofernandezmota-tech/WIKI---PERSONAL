# 3.2 Decoradores

## 🎯 ¿Qué son los Decoradores?

Los **decoradores** son una forma de modificar o extender el comportamiento de funciones o métodos sin cambiar su código.

En Python, los decoradores más comunes para POO son:
- `@property`
- `@staticmethod`
- `@classmethod`

---

## 🔹 @property

### **Problema: Acceso Directo a Atributos**

```python
class Servidor:
    def __init__(self, host, puerto):
        self.host = host
        self.puerto = puerto

# Uso
servidor = Servidor("localhost", 8080)
servidor.puerto = -1  # ❌ Puerto inválido, no hay validación
print(servidor.puerto)  # -1
```

### **Solución Tradicional: Getters y Setters**

```python
class Servidor:
    def __init__(self, host, puerto):
        self.host = host
        self._puerto = puerto  # Atributo "privado"
    
    def get_puerto(self):
        return self._puerto
    
    def set_puerto(self, puerto):
        if puerto < 1 or puerto > 65535:
            raise ValueError("Puerto debe estar entre 1 y 65535")
        self._puerto = puerto

# Uso
servidor = Servidor("localhost", 8080)
print(servidor.get_puerto())  # ❌ Feo
servidor.set_puerto(9000)      # ❌ Feo
```

### **Solución Elegante: @property**

```python
class Servidor:
    def __init__(self, host, puerto):
        self.host = host
        self._puerto = puerto
    
    @property
    def puerto(self):
        """Getter de puerto"""
        return self._puerto
    
    @puerto.setter
    def puerto(self, valor):
        """Setter de puerto con validación"""
        if valor < 1 or valor > 65535:
            raise ValueError("Puerto debe estar entre 1 y 65535")
        self._puerto = valor
    
    @property
    def url(self):
        """Property calculada - solo lectura"""
        return f"http://{self.host}:{self._puerto}"

# Uso
servidor = Servidor("localhost", 8080)
print(servidor.puerto)    # ✅ Elegante (como atributo)
servidor.puerto = 9000    # ✅ Validación automática
print(servidor.url)       # http://localhost:9000
servidor.puerto = -1      # ❌ ValueError
```

### **Ventajas de @property:**
```python
✅ Sintaxis de atributo (servidor.puerto en lugar de servidor.get_puerto())
✅ Validaciones centralizadas
✅ Encapsulación sin sacrificar legibilidad
✅ Puede calcular valores dinámicamente
✅ Properties de solo lectura
```

## 🔸 @staticmethod

### **¿Cuándo usar?**

Cuando necesitas un método que **no requiere acceso a la instancia ni a la clase**.

### **Sin @staticmethod:**
```python
class Utilidades:
    def validar_ip(self, ip):  # ❌ Necesita self innecesariamente
        partes = ip.split(".")
        if len(partes) != 4:
            return False
        return all(parte.isdigit() and 0 <= int(parte) <= 255 for parte in partes)

# Uso
util = Utilidades()
resultado = util.validar_ip("192.168.1.1")  # ❌ Necesitas instancia
```

### **Con @staticmethod:**
```python
class Utilidades:
    @staticmethod
    def validar_ip(ip):
        """Valida formato de dirección IP"""
        partes = ip.split(".")
        if len(partes) != 4:
            return False
        return all(parte.isdigit() and 0 <= int(parte) <= 255 for parte in partes)
    
    @staticmethod
    def validar_puerto(puerto):
        """Valida número de puerto"""
        return isinstance(puerto, int) and 1 <= puerto <= 65535
    
    @staticmethod
    def parsear_url(url):
        """Extrae componentes de URL"""
        # Ejemplo: http://localhost:8080/api
        partes = url.replace("http://", "").replace("https://", "").split("/")
        host_puerto = partes[0].split(":")
        return {
            "host": host_puerto[0],
            "puerto": int(host_puerto[1]) if len(host_puerto) > 1 else 80,
            "path": "/" + "/".join(partes[1:]) if len(partes) > 1 else "/"
        }

# Uso (sin crear instancia)
if Utilidades.validar_ip("192.168.1.1"):
    print("IP válida")

if Utilidades.validar_puerto(8080):
    print("Puerto válido")

componentes = Utilidades.parsear_url("http://localhost:8080/api/usuarios")
print(componentes)  # {'host': 'localhost', 'puerto': 8080, 'path': '/api/usuarios'}
```

## 🔷 @classmethod

### **¿Cuándo usar?**

Cuando necesitas un método que **opera sobre la clase**, no sobre instancias.

### **Ejemplo 1: Constructores Alternativos**
```python
class Configuracion:
    def __init__(self, host, puerto, debug):
        self.host = host
        self.puerto = puerto
        self.debug = debug
    
    @classmethod
    def desde_env(cls):
        """Constructor desde variables de entorno"""
        import os
        return cls(
            host=os.getenv("APP_HOST", "localhost"),
            puerto=int(os.getenv("APP_PORT", "8080")),
            debug=os.getenv("DEBUG", "False") == "True"
        )
    
    @classmethod
    def desarrollo(cls):
        """Configuración para desarrollo"""
        return cls("localhost", 8080, True)
    
    @classmethod
    def produccion(cls, host):
        """Configuración para producción"""
        return cls(host, 80, False)

# Uso
config1 = Configuracion("localhost", 8080, True)  # Constructor normal
config2 = Configuracion.desde_env()  # Desde variables entorno
config3 = Configuracion.desarrollo()  # Preset desarrollo
config4 = Configuracion.produccion("prod-server.com")  # Preset producción
```

### **Ejemplo 2: Factory Pattern**
```python
class ConexionDB:
    def __init__(self, driver, host, puerto, database):
        self.driver = driver
        self.host = host
        self.puerto = puerto
        self.database = database
    
    @classmethod
    def postgresql(cls, host, database):
        return cls("postgresql", host, 5432, database)
    
    @classmethod
    def mysql(cls, host, database):
        return cls("mysql", host, 3306, database)
    
    @classmethod
    def sqlite(cls, archivo):
        return cls("sqlite", "localhost", None, archivo)
    
    def conectar(self):
        return f"Conectando a {self.driver}://{self.host}:{self.puerto}/{self.database}"

# Uso
db_postgres = ConexionDB.postgresql("localhost", "mydb")
db_mysql = ConexionDB.mysql("192.168.1.100", "app_db")
db_sqlite = ConexionDB.sqlite("local.db")

print(db_postgres.conectar())
print(db_mysql.conectar())
print(db_sqlite.conectar())
```

### **Ejemplo 3: Contadores y Estadísticas**
```python
class Peticion:
    total_peticiones = 0
    peticiones_exitosas = 0
    peticiones_fallidas = 0
    
    def __init__(self, url, metodo="GET"):
        self.url = url
        self.metodo = metodo
        self.exitosa = None
        Peticion.total_peticiones += 1
    
    def marcar_exitosa(self):
        self.exitosa = True
        Peticion.peticiones_exitosas += 1
    
    def marcar_fallida(self):
        self.exitosa = False
        Peticion.peticiones_fallidas += 1
    
    @classmethod
    def estadisticas(cls):
        return {
            "total": cls.total_peticiones,
            "exitosas": cls.peticiones_exitosas,
            "fallidas": cls.peticiones_fallidas,
            "tasa_exito": cls.peticiones_exitosas / cls.total_peticiones * 100 if cls.total_peticiones > 0 else 0
        }
    
    @classmethod
    def resetear_contador(cls):
        cls.total_peticiones = 0
        cls.peticiones_exitosas = 0
        cls.peticiones_fallidas = 0

# Uso
p1 = Peticion("/api/usuarios")
p1.marcar_exitosa()

p2 = Peticion("/api/posts")
p2.marcar_exitosa()

p3 = Peticion("/api/error")
p3.marcar_fallida()

print(Peticion.estadisticas())
# {'total': 3, 'exitosas': 2, 'fallidas': 1, 'tasa_exito': 66.67}
```

## 📊 Comparación de Decoradores

| Decorador | Primer Parámetro | Acceso a Instancia | Acceso a Clase | Uso Típico |
|-----------|------------------|-------------------|----------------|------------|
| `@property` | `self` | ✅ Sí | ✅ Sí | Getters/setters |
| `@staticmethod` | Ninguno | ❌ No | ❌ No | Utilidades |
| `@classmethod` | `cls` | ❌ No | ✅ Sí | Constructores alternativos |

## 🎨 Ejemplo Completo: Sistema de Caché

```python
import time
from datetime import datetime, timedelta

class Cache:
    """Sistema de caché en memoria"""
    
    _instancia = None
    instancias_creadas = 0
    
    def __init__(self, ttl_segundos=300):
        self._cache = {}
        self._ttl_segundos = ttl_segundos
        Cache.instancias_creadas += 1
    
    @property
    def size(self):
        """Tamaño actual del caché (solo lectura)"""
        return len(self._cache)
    
    @property
    def ttl_segundos(self):
        """TTL en segundos"""
        return self._ttl_segundos
    
    @ttl_segundos.setter
    def ttl_segundos(self, valor):
        """Setter con validación"""
        if valor <= 0:
            raise ValueError("TTL debe ser positivo")
        self._ttl_segundos = valor
    
    @staticmethod
    def generar_key(prefijo, identificador):
        """Genera clave del caché"""
        return f"{prefijo}:{identificador}"
    
    @staticmethod
    def _es_expirado(timestamp, ttl):
        """Verifica si timestamp está expirado"""
        return datetime.now() > timestamp + timedelta(seconds=ttl)
    
    @classmethod
    def get_instance(cls):
        """Patrón Singleton"""
        if cls._instancia is None:
            cls._instancia = cls()
        return cls._instancia
    
    @classmethod
    def total_instancias(cls):
        return cls.instancias_creadas
    
    def set(self, key, valor):
        """Guardar en caché"""
        self._cache[key] = {
            "valor": valor,
            "timestamp": datetime.now()
        }
    
    def get(self, key):
        """Obtener de caché"""
        if key not in self._cache:
            return None
        
        entrada = self._cache[key]
        
        # Verificar si expiró
        if self._es_expirado(entrada["timestamp"], self._ttl_segundos):
            del self._cache[key]
            return None
        
        return entrada["valor"]
    
    def limpiar(self):
        """Limpiar caché completo"""
        self._cache.clear()
    
    def limpiar_expirados(self):
        """Limpiar solo elementos expirados"""
        keys_expiradas = [
            key for key, entrada in self._cache.items()
            if self._es_expirado(entrada["timestamp"], self._ttl_segundos)
        ]
        for key in keys_expiradas:
            del self._cache[key]
        return len(keys_expiradas)

# Uso
cache = Cache(ttl_segundos=60)

# @property
print(f"Tamaño: {cache.size}")  # 0
cache.ttl_segundos = 120  # Cambiar TTL

# @staticmethod
key_usuario = Cache.generar_key("user", 123)
print(key_usuario)  # "user:123"

# Guardar y recuperar
cache.set(key_usuario, {"nombre": "Alvaro", "email": "alvaro@example.com"})
usuario = cache.get(key_usuario)
print(usuario)  # {'nombre': 'Alvaro', 'email': 'alvaro@example.com'}

print(f"Tamaño: {cache.size}")  # 1

# @classmethod - Singleton
cache2 = Cache.get_instance()
print(cache2 is cache)  # False (diferentes instancias)
print(Cache.total_instancias())  # 2
```

## ✅ Cuándo Usar Cada Decorador

### **@property:**
```
✅ Validar al asignar valores
✅ Calcular valores dinámicamente
✅ Atributos de solo lectura
✅ Encapsulación sin complicar sintaxis
✅ Propiedades computadas
```

### **@staticmethod:**
```
✅ Funciones auxiliares relacionadas con la clase
✅ Validaciones independientes
✅ Parseos y conversiones
✅ No necesita ni self ni cls
✅ Utilidades generales
```

### **@classmethod:**
```
✅ Constructores alternativos (factory methods)
✅ Operaciones sobre la clase (contadores, estadísticas)
✅ Patrones Singleton o Factory
✅ Necesita acceso a cls
✅ Métodos que retornan instancias
```

---

## 📝 Resumen

- **@property** = Getters/setters elegantes con validación
- **@staticmethod** = Métodos sin self ni cls (utilidades)
- **@classmethod** = Métodos que operan sobre la clase
- **Combinación** = Clases robustas y profesionales
