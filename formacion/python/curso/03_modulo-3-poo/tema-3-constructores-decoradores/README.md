# 📦 Tema 3 - Constructores y Decoradores

## 🎯 Objetivo

Dominar el uso de constructores (`__init__`) y decoradores (`@property`, `@staticmethod`, `@classmethod`) en Python para crear clases más robustas y profesionales.

---

## 📚 Contenido

### **3.1 Constructores**
- Constructor `__init__`
- Parámetros del constructor
- Valores por defecto
- Validaciones en el constructor
- Constructores alternativos con `@classmethod`

### **3.2 Decoradores**
- `@property` - Getters y setters elegantes
- `@staticmethod` - Métodos sin instancia
- `@classmethod` - Métodos de clase
- Decoradores personalizados (introducción)

### **3.3 Bonus - Buenas Prácticas**
- Principios SOLID aplicados
- Cuándo usar cada decorador
- Patrones de diseño básicos
- Code smells comunes
- Refactoring de código
