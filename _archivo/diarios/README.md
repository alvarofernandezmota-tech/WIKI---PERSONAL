## 📌 Convenciones

- `YYYY-MM-DD.md` — diario general del día
- `YYYY-MM-DD-descripcion.md` — sesión específica
- `_plantilla.md` — plantilla para nuevos diarios
- Carpeta `archivo-jun2026/` — diarios archivados
- Carpeta `2026/` — organización por año (en progreso)
