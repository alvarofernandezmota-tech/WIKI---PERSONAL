# 🤖 Isla: IA Local

> Los modelos corriendo y la memoria del ecosistema.
> Base de todo lo que usa inteligencia artificial.

---

## Repos

### [`ollama-stack`](https://github.com/alvarofernandezmota-tech/ollama-stack) 🔒 Privado
Infraestructura de modelos LLM locales en Madre:
- **Ollama** — ejecuta los modelos localmente
- **Open WebUI** — interfaz web para chat con modelos
- **LiteLLM** — gateway unificado de APIs
- Sirve modelos a THDORA y a todo el ecosistema

### [`local-brain`](https://github.com/alvarofernandezmota-tech/local-brain) 🔒 Privado
Memoria y conocimiento vectorial del ecosistema:
- **Qdrant** — base de datos vectorial
- **pgvector** — vectores en PostgreSQL
- Pipelines de RAG (Retrieval-Augmented Generation)
- Embeddings de documentación y diarios
