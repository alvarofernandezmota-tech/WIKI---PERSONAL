# 🌱 Isla: Vida Personal

Esta isla es el origen. Antes de que existiera el ecosistema técnico, existía el diario.
`VIDAPERSONAL` es el repo donde vive todo lo personal — diarios, metas, reflexiones, hábitos, formación personal.

> 📓 Repo canónico → [`VIDAPERSONAL`](https://github.com/alvarofernandezmota-tech/VIDAPERSONAL)

---

## Qué hay en VIDAPERSONAL

| Carpeta | Contenido |
|---|---|
| `00_yo` | Quién eres, valores, identidad |
| `00_sistema` | Cómo funciona tu sistema personal |
| `01_diarios` | Diario personal |
| `01_traking_diario` | Tracking de hábitos y días |
| `02_formacion` | Aprendizaje personal (conectado con isla Formación) |
| `02_metas` | Objetivos y metas |
| `03_proyectos` / `05_proyectos` | Proyectos personales |
| `03_reflexiones` | Reflexiones y pensamientos |
| `04_curiosidad` | Ideas, investigaciones, cosas que te llaman la atención |
| `05_contenido` | Contenido que produces o consumes |
| `inbox/` | Entrada sin clasificar |

## Relación con el ecosistema técnico

- `02_formacion` en VIDAPERSONAL → conectado con [`wiki/islas/formacion.md`](./formacion.md)
- El diario personal es independiente de los diarios técnicos de Dew
- Los proyectos personales pueden convertirse en islas del ecosistema cuando crecen

## Regla de separación

| VIDAPERSONAL | WIKI + Dew |
|---|---|
| Diario personal, emociones, vida | Ecosistema técnico, decisiones de arquitectura |
| Metas personales | Objetivos técnicos (issues, HAL) |
| Reflexiones | ADRs y diarios de sesión |
| Formación personal | Plan de aprendizaje técnico |

Son el mismo cerebro — dos contextos distintos.
