## Repos

### [`dev-labs`](https://github.com/alvarofernandezmota-tech/dev-labs) 🔒 Privado
Sandbox de desarrollo del ecosistema:
- Prototipos web rápidos
- Scripts sueltos y utilidades CLI
- Experimentos de código sin repo propio
- Pruebas antes de escalar a producción

---

## Criterio para salir de Labs

Un proyecto pasa de `dev-labs` a su propio repo cuando cumple **alguno** de estos:

```
✅ Más de 20 commits propios
✅ Tiene deploy independiente
✅ Otros proyectos lo importan como dependencia
✅ Necesita su propio CI/CD
```
