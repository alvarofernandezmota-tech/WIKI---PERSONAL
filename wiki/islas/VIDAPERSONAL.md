# 🌱 Isla VIDAPERSONAL

> Repo privado donde vive todo lo personal: planificación, diarios, hábitos, reflexiones, proyectos de vida.
> El DEW ejecuta. La WIKI documenta. VIDAPERSONAL planifica y registra.

---

## Rol en el ecosistema

```
        VIDAPERSONAL
        (planificas, diarios,
         hábitos, reflexiones)
              ▲
             / \
            /   \
      DEW ◄───────► WIKI
  (ejecutas,        (mapas,
   issues, CI)       conceptos)
```

**Regla de oro:** Si es accionable con fecha → DEW (issue). Si es reflexión, registro o planificación → VIDAPERSONAL. Si es concepto o mapa de conocimiento → WIKI.

## Canon semanal

El archivo `docs/canon/ESTADO-VIDA.md` se actualiza **cada domingo** con:
- Rachas de hábitos con datos reales
- Metas 2026 con % de progreso
- Proyectos activos
- Estado emocional y energía
