# 🤖 Perplexity — contexto y uso

## Qué es
Perplexity es el agente IA principal con acceso a este repo vía MCP.
Actualiza GitHub directamente sin pasar por local.

## Cómo trabaja con la wiki
- Lee `CONTEXT.md` para entender el ecosistema
- Lee `AGENT.md` para saber cómo comportarse
- Lee `HOME.md` para orientarse
- Puede crear/editar ficheros vía MCP

## Regla importante
Cuando Perplexity hace cambios en GitHub, Madre no los tiene en local.
Siempre hacer `git pull` al empezar una sesión en Madre.

## Casos de uso
- Actualizar diarios
- Crear estructura de ficheros
- Documentar cambios del ecosistema
- Responder preguntas con contexto del repo
