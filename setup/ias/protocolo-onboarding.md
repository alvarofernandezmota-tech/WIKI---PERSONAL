# Protocolo Onboarding — Nuevo Agente IA
*Creado: 24 junio 2026*

> Cada vez que usemos un modelo IA nuevo o una versión nueva, seguimos este protocolo antes de usarlo para trabajo real.

---

## REGLA

**Antes de usar cualquier IA para investigación o trabajo — entrevístala primero.**

Esto nos permite:
- Saber exactamente con qué versión estamos trabajando
- Conocer sus límites reales (no los del marketing)
- Detectar sesgos antes de que contaminen el trabajo
- Tener documentación viva de cada agente
- Comparar modelos de forma justa

---

## PASOS

1. **Abre un chat nuevo** con el modelo
2. **Pega el PROMPT DE AUTOCONOCIMIENTO** de `PLANTILLA-AGENTE.md`
3. **Recoge la respuesta** y tráela a Perplexity
4. **Perplexity documenta** en `setup/ias/[modelo].md`
5. **Evalúa** con la tabla de criterios
6. **Ya puedes usarlo** para trabajo real
