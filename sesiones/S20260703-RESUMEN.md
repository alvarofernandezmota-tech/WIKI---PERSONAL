## ⚠️ Problemas detectados esta sesión

### 1. SSH key de madre no tiene permiso en GitHub
```
git@github.com: Permission denied (publickey)
fatal: Could not read from remote repository.
```
**Fix:** Añadir la clave pública de madre a GitHub → Settings → SSH Keys
```bash
# En madre:
cat ~/.ssh/id_ed25519.pub
# Copiar y pegar en https://github.com/settings/keys
```

### 2. gh CLI no autenticado en madre
```
(gh CLI no autenticado o sin issues abiertos)
```
**Fix:**
```bash
gh auth login
```

### 3. Blink se desconectó por socket timeout
- `Session Exception - Socket error: Socket is not connected`
- Normal en sesiones largas — reconectar con `ssh madre`
- **Mitigación:** Añadir a `~/.ssh/config` en iPhone:
  ```
  Host madre
    ServerAliveInterval 60
    ServerAliveCountMax 3
  ```
