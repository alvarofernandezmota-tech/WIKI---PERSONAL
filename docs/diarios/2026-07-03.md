## Notas técnicas

### Sobre separar repos
Regla: separar cuando una carpeta tiene ciclo de vida, releases o equipo propio.
Candidatos actuales: `osint-stack/` (creciendo), `thdora/` (Fase 7).
Cuando se separen, heredan `docs/filosofia/FILOSOFIA.md` y convenciones.

### Sobre Docker skills compartidos
Cuando un servicio crece a repo propio, comparte via:
- `docker/shared/` — redes y volúmenes comunes
- `.env.template` — variables compartidas
- `scripts/` — utilidades comunes como submódulo o copia documentada
