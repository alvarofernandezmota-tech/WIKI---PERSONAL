## Aprendizajes

- Blink Shell es superior a Shellfish para SSH en iOS — mejor copiar/pegar
- El campo `Key` en Blink hosts es crítico — sin él usa el agente y falla
- `git pull --no-rebase` antes de push cuando hay commits remotos nuevos
- Tailscale + SSH ed25519 = doble capa sólida sin abrir puertos al exterior
- `ssh-add ~/.ssh/id_ed25519_github` evita escribir passphrase en cada push
