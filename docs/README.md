# 📖 Docs

Documentación técnica permanente del ecosistema.

## Estructura

```
docs/
├── ecosistema/   ← mapas, guías de continuación
└── infra/        ← accesos, red, hardware
```

## Diferencia con wiki/
- `docs/` — documentación técnica densa, de referencia
- `wiki/` — notas operativas, guías rápidas, runbooks
