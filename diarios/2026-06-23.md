## 🗺️ El gran descubrimiento — La Batcueva

Hoy quedó claro el concepto: el **dashboard no es un proyecto secundario, es la capa central** que une todo el ecosistema. Cada herramienta OSINT/monitorización corre en su puerto y el dashboard las embebe.

### Stack definitivo — 5 herramientas OSINT

| Herramienta | Capa | Qué hace |
|---|---|---|
| **OSIRIS** | Mundo físico global | Globo 3D: 10k+ aviones, 2000+ satélites, cámaras, seísmos, nuclear |
| **Shadowbroker** | Aviones + barcos + GPS | Vuelos militares, 25k barcos AIS, GPS jamming en tiempo real |
| **SpiderFoot** | OSINT digital | IPs, dominios, emails, redes sociales, personas |
| **Kismet** | Red local | WiFi, Bluetooth, dispositivos físicos alrededor |
| **Maltego** | Conexiones y grafos | Visualiza relaciones entre personas/IPs/empresas |

> Shodan no necesita instalación — se usa desde la web (gratis).

### El flujo real
```
VES avión sospechoso en Shadowbroker
    → buscas matrícula/dueño en SpiderFoot
    → localizas IP del aeropuerto en Shodan
    → OSIRIS te da satélites + cámaras CCTV del área
    → Kismet detecta dispositivos WiFi cerca de ti
```
