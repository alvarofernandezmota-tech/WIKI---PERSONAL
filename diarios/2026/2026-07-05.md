## Aprendido
- GitHub NO se actualiza solo → siempre `git push`
- MCP actúa directo en GitHub → siempre `git pull` al empezar sesión
- Wiki = mapa + docs propios. NO duplica contenido de otros repos
- Cada repo tiene su propio README y docs internos
- La wiki enlaza, no contiene
- Workflows = orquestadores | Actions = piezas | Bots = apps externas
- Siempre sincronizar al cerrar sesión
