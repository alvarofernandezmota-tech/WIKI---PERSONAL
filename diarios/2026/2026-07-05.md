## Aprendido
- GitHub NO se actualiza solo, necesita `git push`
- MCP actúa directamente sobre GitHub sin pasar por local → siempre hacer `git pull` al empezar
- Workflows = orquestadores | Actions = piezas reutilizables | Bots = apps externas
- Siempre sincronizar al final de sesión: `git add . && git commit && git push` en cada repo tocado
