## ❌ Bloqueantes encontrados

| Problema | Causa | Solución |
|---|---|---|
| `adb input keyevent` denegado | Android 13 MIUI bloquea `INJECT_EVENTS` sin root | Usar `adb shell monkey` para apps |
| `adb tcpip 5555` bloqueado vía red | Android no expone puerto 5555 por defecto | Solo funciona por USB |
| GitHub bloquea curl APKs | Rate limit / redirect | F-Droid como fuente alternativa |
| `INSTALL_FAILED_USER_RESTRICTED` | MIUI requiere "Instalar vía USB" + verificación Mi | Activar opción + verificar |
| Docker pull Kali cortado x2 | Caída de red | Usar `tmux` siempre |

## 🧠 Aprendizajes

- **Android 13 bloquea `input keyevent`** sin root — usar `monkey` para abrir apps
- **F-Droid** es fuente fiable para APKs open source cuando GitHub bloquea
- **Siempre tmux** en madre para procesos largos — la red cae
