### PASO 1 — Hotspot: fijar 5GHz y evitar desconexión 📶

> Investigación hecha el 23 jun. Ver archivos en inbox para aplicar.
> Objetivo: hotspot WiFi que no se desconecte nunca + forzar banda 5GHz para más velocidad.

**En Android (móvil que hace de hotspot):**
```
Ajustes → Punto de acceso → Configuración avanzada:
  ✔ Banda: 5 GHz (forzar, no “automático”)
  ✔ Apagar automáticamente: DESACTIVAR
  ✔ Compatible con dispositivos más antiguos: OFF (libera banda 5GHz pura)
```

**Con ADB (si no aparece la opción en UI):**
```bash
# Conectar ADB al móvil
adb devices

# Forzar banda 5GHz
adb shell settings put global wifi_hotspot_band 2
# 0 = auto, 1 = 2.4GHz, 2 = 5GHz

# Desactivar apagado automático
adb shell settings put global hotspot_no_internet_tethering_disabled 1

# Verificar
adb shell settings get global wifi_hotspot_band
```

**Verificar banda activa desde madre:**
```bash
nmcli dev wifi list | grep -i 5
iwconfig 2>/dev/null || iw dev
```

- [ ] Verificar que madre ve la red en 5GHz
- [ ] Confirmar que no se cae con `ping -i 5 madre` durante 2 min
