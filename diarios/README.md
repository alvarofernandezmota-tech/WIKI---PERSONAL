# 📓 Diarios técnicos

Log cronológico de sesiones de trabajo técnico. Qué se hizo, qué se rompió, qué se aprendió.

## Estructura

```
diarios/
├── YYYY/
│   └── YYYY-MM-DD.md   ← una entrada por día
```

## Convención de entrada

Cada fichero `YYYY-MM-DD.md` sigue la plantilla:
```
# 📅 YYYY-MM-DD

## Qué hice
## Qué rompí / problemas
## Qué aprendí
## Pendiente para mañana
```

## Dónde van los diarios personales
Los diarios de vida personal (reflexiones, objetivos, salud) van en el repo **VIDAPERSONAL**, no aquí.
